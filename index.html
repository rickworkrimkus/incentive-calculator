Rimkus Incentive Engine v3

Client-facing screening calculator for decarbonization funding, incentive stacking, annual utility savings, greenhouse gas reduction, and Version 3 NOI / cap-rate impact modelling. This version adds valuation assumptions so teams can connect savings and incentives to asset value strategy.

This remains a screening tool. Final incentive values, utility impacts, and eligibility must be confirmed through engineering review, program validation, and actual site conditions.

4.5) NOI and cap-rate assumptions

Use this section to translate recurring savings into operating income and value impact. Incentives reduce net capital, while recurring utility and maintenance savings can support NOI uplift and implied asset value creation.

4) Incentive stacking engine

Each program card can be enabled, assigned a funding type, and given its own % assumptions, cap, eligibility category, and stacking behavior. The model applies caps, then multiplies by stacking compatibility and category relevance. Conservative, expected, and aggressive scenarios are generated automatically.

Version 2 logic now distinguishes between broad funding, gas-centric incentives, electric incentives, water incentives, and financing/performance funding. This creates a more realistic range than simply adding all percentages together.
